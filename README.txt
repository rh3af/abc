Download Anaconda for your OS from: https://www.anaconda.com/products/distribution#Downloads

If anaconda is already installed and not updated run commands: 
conda update conda
conda update anaconda

Step 1: Open Anaconda Prompt (anaconda3) on Windows | Terminal on MacOS.

Step 2: Mount the extracted "biomed_final.zip".

Step 3: For Windows type command "windows_env.bat" on Anaconda Prompt to setup the environment.

Step 4: For MacOS type command "source mac_env.sh" on terminal to setup the environment.

Step 5: on "./biomed_final/src/" run command "python run.py" (Takes hours to execute).

Step 6: on "./biomed_final/" run command "python model_evaluation.py".
